## 童年回忆

最早接触小霸王的时候我好像还没上小学也才刚开始记事，在表哥家里有这个游戏机。当时我最喜欢玩的就是超级玛丽，但我每次都碰不到终点的那个旗子。问我哥他就是不跟我说，每次还故意把按键挡住了不给我看。没记错的话，这是我最早接触的一款游戏了。
